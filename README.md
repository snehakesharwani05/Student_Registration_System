# 🎓 Student Registration System

A modern **Student Registration System** built with **Django** and deployed on **Amazon Web Services (AWS)** to demonstrate secure cloud-based application hosting and database management. The application is hosted on an **Amazon EC2** instance with a public URL, while all student records are securely stored in an **Amazon RDS (MySQL)** database.

This project showcases the practical implementation of key AWS services, including **EC2** for application hosting, **Amazon RDS** for managed database services, **Amazon EBS Volumes** and **Snapshots** for data backup and recovery, and an **Amazon S3 Bucket** for storing the project source code and application screenshots.

### 🚀 Features

* Student Registration, Update, Search & Delete
* Secure MySQL Database using Amazon RDS
* Live Deployment on Amazon EC2
* EBS Volume & Snapshot for Backup and Recovery
* Amazon S3 for Project Assets and Documentation
* Public EC2 URL for Live Application Access

### 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3
* **Backend:** Django, Python
* **Database:** MySQL (Amazon RDS)
* **Cloud Services:** Amazon EC2, Amazon RDS, Amazon EBS, EBS Snapshots, Amazon S3
* **Tools:** Git, GitHub, VS Code
