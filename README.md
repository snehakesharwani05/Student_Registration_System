# Student_Registration_System

A cloud-based **Student Registration System** built with **Django** and **MySQL (Amazon RDS)** for secure student data management. The application is deployed on an **Amazon EC2** instance with a publicly accessible URL, while student records are stored in a protected **AWS RDS** database. The project also demonstrates core AWS services, including **EBS Volumes**, **Snapshots** for backup and recovery, and an **Amazon S3 Bucket** for storing the project source code and application screenshots.

**Tech Stack:** Django, Python, HTML/CSS, MySQL, AWS EC2, Amazon RDS, Amazon EBS, Snapshots, Amazon S3.
